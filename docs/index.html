Features

它会做的事

像一张唱片的曲目单——每一条都开箱即用，不需要你配置任何东西。

01歌词自动来本地 LRC → 在线曲库秒出 → 本机 AI 识别兜底；系统引擎先出草稿，Whisper 后台精修，屏幕永远不空等。
02逐字流动歌词Apple Music 式卡拉 OK：当前行放大、逐词点亮、自动滚动，点任意一行跳转播放，可导出 LRC。
03踩鼓点的光晕窗口边缘的动态光晕按整曲预分析的拍点网格零延迟弹跳，还会洒到窗外；视频时换成画面氛围色。不喜欢可一键关。
04视频也体面原生悬浮控制条、窗口贴合视频分辨率、界面自动隐身、字幕自动识别叠加。
05会议它来记实时字幕浮窗边听边转文字，还能分辨谁在说话——在场几人、每人说了什么，可导出记录与录音。
06格式通吃mp3 / flac / m4a / wav / mp4 / mov 原生播放；mkv / webm / ogg / opus / ape / avi 等经 FFmpeg 解锁。

Install

安装教程

Echo Player 是独立开发的开源应用，没有加入 Apple Developer 计划（99 美元/年），所以首次打开需要多点两下——只有第一次，之后的自动更新完全无感。

下载并解压

点上方「下载最新版」得到 zip，双击解压出 Echo Player.app，把它拖进「应用程序」文件夹。

首次打开：右键 → 打开

不要直接双击。右键点击应用图标，选「打开」，在弹窗里再点一次「打开」。这是 macOS 对未签名应用的例行确认。

如果系统仍然拦截

打开 系统设置 → 隐私与安全性，拉到底部找到「已阻止 Echo Player」，点「仍要打开」。或者在终端执行：xattr -cr /Applications/Echo\ Player.app

授权麦克风（可选）

只有用「实时字幕 / 会议记录」功能时才会请求麦克风权限；所有识别都在本机完成，音频不会上传。

为什么之后的更新不用再折腾？应用内置更新器：发现新版 → 应用内下载（校验 SHA-256 指纹防篡改）→ 自动替换重启。应用内下载不带隔离标记，Gatekeeper 不会再拦。
