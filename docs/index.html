音频不出 Mac

第一次打开，多点两下。

没有 Apple 签名，macOS 会多问一次。

1解压，拖进「应用程序」。
2右键 → 打开，再点一次「打开」。
3还被拦？系统设置 → 隐私与安全性 → 「仍要打开」。或 xattr -cr /Applications/Echo\ Player.app

之后的更新在应用内完成，不再被拦。
