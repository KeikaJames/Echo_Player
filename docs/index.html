音频不出你的 Mac

第一次打开，多点两下。

独立开发，没有 Apple 签名——macOS 会多问一次，只有这一次。

1解压后把 Echo Player 拖进「应用程序」。
2右键点它，选「打开」，弹窗里再点一次「打开」。
3还被拦？系统设置 → 隐私与安全性 → 「仍要打开」。或执行 xattr -cr /Applications/Echo\ Player.app

之后的更新在应用内完成（SHA-256 校验），不会再被拦。
